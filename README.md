# python-100days
Pythonの100日チャレンジ 🌱

家庭菜園の趣味とPythonを組み合わせた100日学習記録です。

##　作っているもの
野菜の月別作業チェッカー（栽培管理アプリ）

##　進捗
Day 1-2: 変数、データ型、演算子の復習
　　料理レシピの材料計算機(人数に応じて分量調整)
Day 3-4: 条件分岐(if/elif/else)
　　野菜の栽培可能時期判定プログラム(月を入力→今植えられる野菜を提案)
Day 5-6: ループ(for/while)
　　1週間分の献立リストジェネレーター
Day 7-8: リスト、辞書、タプル
　　家庭菜園の作物管理リスト(辞書で品種・植え付け日・収穫予定日を管理)
Day 9-10: 関数の基本
　　BMI計算&健康アドバイス関数作成
